# Miscelanious tools project details

This document describes the electronic construction of this project.

## Electronic componets

- 1x Esspresif ESP32.
- 1x audio amplifier for 5v and 3W output.
- 1x 5v tension stabilizer module.
- 1x 8ohms speaker.
